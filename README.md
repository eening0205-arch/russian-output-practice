# Russian Output Practice

Static Russian speaking-output practice project.

The first version trains daily Russian expressions from Chinese prompts. Standard Russian answers and audio stay hidden until the learner clicks `显示答案`.

## Sources

- Weekly practice packs live in `weekly_packs/`.
- The current vocabulary source is the eyujun export at `/Users/weiwei/Ning/outputs/eyujun/nin-knig-words-after-daily-notes-2026-06-09.csv`.
- Apple Notes vocabulary scans are used through the existing eyujun output files, not by reading unrelated notes.
